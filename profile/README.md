## 🚀 关于我们 / About Us
**Infused Synapse AI** 是一家专注于**具身智能 (Embodied AI)** 的前沿技术实验室。我们致力于通过深度神经网络与先进的运动控制算法，赋予机器人在复杂物理世界中感知、决策与执行的能力。

### 🛠️ 技术栈 / Tech Stack
* **软件框架**: ROS2 (Humble), Python, C++17
* **感知系统**: Livox Mid-360 LiDAR, 多模态视觉融合
* **智能平台**: NVIDIA Jetson AGX Orin, RealMan RMC-DA
